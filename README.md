## Getting Started
### Prerequisites

 - [Node.js](https://nodejs.org/es/)

### Usage

You can clone this repsitory to get the source code

    git clone https://github.com/vergaraldvm/Unetz.git

Install dependencies

    npm install

Run the application

    npm start
